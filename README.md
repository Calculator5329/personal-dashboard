# Personal Dashboard

A Google Calendar-integrated dashboard for managing events and financial planning with Google OAuth authentication.

## What It Is

Personal Dashboard is an integrated scheduling and financial planning tool that syncs with your Google Calendar. It provides a calendar interface for viewing upcoming events, managing appointments, and setting up recurring financial events. The app uses Google OAuth for authentication and includes tools for financial event creation and tracking.

## Features

- **Google Calendar Integration** - View, create, and manage events directly from Google Calendar
- **Calendar View** - Interactive calendar interface with event management
- **Event Management** - Create, update, and delete events from the dashboard
- **Financial Planning** - Setup recurring financial events and balance tracking
- **Google OAuth** - Secure authentication using Google accounts
- **Event Modal** - Intuitive modal for creating and editing events
- **Date Management** - Navigate between months with ease

## Tech Stack

- React 19 with TypeScript
- Vite for development and building
- Google Calendar API integration
- Google OAuth for authentication
- date-fns for date manipulation
- React hooks for state management

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```
